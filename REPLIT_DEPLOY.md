# Replit Deployment

This project is ready to deploy on **Replit** with one click!

## **INSTANT DEPLOY:**

1. Go to: https://replit.com/github/Mayaaids/Dashboard--backend--main
2. Click **Import from GitHub**
3. Replit auto-deploys your repo
4. Set Run Command to:
   ```
   cd Dashboard--backend--main/symposium-backend && npm start
   ```
